# Darbic

Darbic is a web-based crowd guidance platform for large-scale events. It helps visitors move more efficiently inside event venues by providing assigned entry gates, pickup/drop-off points, and recommended arrival times based on ticket information.

The system aims to reduce congestion, improve safety, and enhance the visitor experience. It also supports special assistance requests such as Golf Cart transportation for visitors with mobility needs.

## Features
### Visitor
- Sign up / Log in
- Enter ticket number
- View assignment details
- Request special assistance
- Update profile
- Receive notifications
- View event history

### Administrator
- Log in / Log out
- Create events
- Manage zones and capacities
- Manage time slots
- Monitor crowd distribution
- Update visitor assignments
- Send notifications
